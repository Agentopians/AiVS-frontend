  7. it's a bit tricky.
  9. there's a big gradient background, that spans the entire window
  10. the white background you see is actually being added on each message
  11. the bubble shape itself is not a blue bubble, but a white border with a transparent center
  13. Effectively, each message is a "window" through which one can take a peek at the background
  15. A little trick is used to give the bubble shape to the messages.
  16. they have an svg image as a border
  17. using the (little known?) "border-image-source" and "border-image-slice" css properties

  23. fixed background makes the browser redraw things every time it scrolls
  24. while doing it this way makes the browser do only a composite operation
